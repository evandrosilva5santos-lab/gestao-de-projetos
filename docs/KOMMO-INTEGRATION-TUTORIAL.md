## Configurando a API

1. Obtenha o seu Subdomínio Kommo (ex: `sua-agencia.kommo.com` -> `sua-agencia`).
2. Obtenha o seu Token de Longa Duração nas integrações do Kommo.
3. No painel, vá em **Central de Integrações** > **Adicionar Integração** > **Kommo CRM**.
4. Insira os dados acima para salvar a conexão.
